"Static" Static positions as per html flow. These elements are considered to be not positioned.
"Relative-1" Relative position is the same as static unless you give it extra values.
"Relative-2" Giving values to top, right, bottom, and left will move the element. These values are relative to its default position.
"Static-1/Absolute" Behaves like relative except that it is relative to its parent element. The parent element needs to have a position:relative. Top, right, bottom, and left give it a new position.
Fixed position will move to a new position and stay there even if content around it is being scrolled through.
